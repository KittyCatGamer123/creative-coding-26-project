# Computational Arts Assignment 2026
*"Create an "Interactive Experience" using Godot entitled: "A Day in the Life of a First Year Game Design Student". I.e. YOU"*

This is a repository dedicated to the Computational Arts assignment I was to do for the second semester of college, which required us to develop a game centred around a "Day in the Life" of my life. 

We were required to make this game in the Godot Engine, using both elements of 2D and 3D systems, and to design our assets and sounds by ourselves. All code was written in GDScript and was to be published on Itch.io and Git.

**This project is still a work-in-progress. If you're reading this, hi!!! :D**

## Notice
- All of the code used in this project is human-written - artificial intelligence (AI) was not used at any point during development of the program.
